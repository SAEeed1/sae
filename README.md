# AI Products Marketplace

یک سایت حرفه‌ای برای فروش محصولات هوش مصنوعی

## محصولات

- 🤖 ایجنت‌های هوشمند (Intelligent Agents)
- 💬 چت‌بات‌های تخصصی (Specialized Chatbots)
- ⚙️ خدمات اتوماسیون (Automation Services)
- 🎬 محصولات رسانه‌ای هوش مصنوعی (AI Media Products)
- 🔧 اسکریپ‌های تخصصی (Custom Scripts)
- 🎯 LLM Orchestration

## ویژگی‌ها

✨ Modern UI/UX Design  
🎨 Smooth Animations (Framer Motion)  
📱 Fully Responsive  
👥 Multi-Admin Support  
📊 Professional Admin Dashboard  
📝 Order Request Form  
🔐 Secure Authentication  
💾 Database Integration  

## نیازمندی‌ها

- Node.js 18+
- npm or yarn

## نصب و اجرا

```bash
npm install
npm run dev
```

سایت در `http://localhost:3000` اجرا خواهد شد.

## ساختار پروژه

```
app/

├── products/
```
